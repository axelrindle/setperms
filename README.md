# setperms
> Two-in-one solution for chmod(1) and chown(1)

## Install
```bash
$ npm install -g setperms
```
or
```bash
$ yarn global add setperms
```

## Usage

```bash
$ setperms [options] <files/dirs...>
```

For help type:
```bash
$ setperms --help
```
